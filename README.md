# AI Career Recommendation System

## Project Description
The AI Career Recommendation System is a machine learning based web application that predicts suitable career paths for students based on their skills, interests, and academic performance.

The system analyzes multiple factors such as programming skills, communication ability, logical reasoning, mathematics performance, and creativity to recommend the most suitable career options.

## Features
- AI based career prediction
- Machine learning model trained on career dataset
- Interactive web interface
- FastAPI backend for prediction
- React frontend for user interaction

## Technologies Used

Frontend
- React
- Vite
- TypeScript
- Tailwind CSS

Backend
- FastAPI
- Python
- Scikit-learn

Machine Learning
- Random Forest
- Gradient Boosting
- XGBoost

## Dataset
The dataset contains 10,000 records and 25 attributes describing student skills, interests, academic scores, and personality traits.

## How to Run the Project

### Step 1: Clone the repository
git clone https://github.com/vaishnavib2007/AI-Career-Recommendation-System.git

### Step 2: Navigate to the project folder
cd AI-Career-Recommendation-System

### Step 3: Install frontend dependencies
npm install

### Step 4: Start the frontend
npm run dev

### Step 5: Start the backend
cd backend
uvicorn app:app --reload

### Step 6: Live Website
ai-career-recommendation-system-61mhy4zt4.vercel.app
